Nyeta Project Review

Road to Production

🐛 Critical Bugs (ต้องแก้ไขทันที)

  • ไมค์และ AI พูดซ้อนกัน: ต้องปิดเสียงพูด AI ก่อนเริ่มรับเสียงผู้ใช้เสมอ เพื่อป้องกัน Feedback Loop
  • Camera Deny UI: แอปค้างเมื่อผู้ใช้ไม่อนุญาตสิทธิ์กล้อง ต้องเพิ่มปุ่มลองใหม่และเสียงแจ้งเตือน
  • Error Boundary Silence: เมื่อระบบล่ม (React Crash) ไม่มีเสียงอ่านสำหรับ Screen Reader
  • Network Drop Handling: ไม่มีการแจ้งเตือนด้วยเสียงเมื่ออินเทอร์เน็ตหลุด

✨ Missing Features (ฟีเจอร์ที่ขาดหาย)

  • ปุ่มหยุดเสียง AI Global: ให้ผู้ใช้สามารถกดหยุดเสียง AI ที่กำลังอธิบายยาว ๆ ได้ในโหมดผู้ช่วย
  • Push Notification: เพิ่มการแจ้งเตือนอาสาสมัครแม้ว่าพับหน้าจอหรือแอปอยู่เบื้องหลัง
  • Landscape Mode: จัดการและล็อก Layout ให้ถูกต้องเมื่อหมุนหน้าจอ

🔒 Security & Performance

  • ซ่อน API Key (สำคัญที่สุด): ต้องย้ายการเรียก Gemini/Groq API ไปไว้ที่ฝั่ง Server-side
  • Progressive Web App: ใช้ Service Worker เพื่อแคชไฟล์ระบบและโมเดล AI ให้ทำงานออฟไลน์และโหลดเร็วขึ้น
  • Battery Management: ลด Framerate กล้อง และหยุดพักโมเดล Object Detector เมื่อผู้ใช้ไม่มีการเคลื่อนไหวเครื่อง

🎯 Priority Action Plan

  • Priority 1: ย้าย API Key ไป Server เพื่อความปลอดภัยของระบบ
  • Priority 2: จัดการปัญหาสายหลุดและ Error พร้อมเสียงแจ้งเตือน
  • Priority 3: เพิ่มระบบทดสอบอัตโนมัติ (Unit & E2E Tests) เพื่อป้องกันบั๊กสะสม
