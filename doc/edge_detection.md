# ระบบตรวจจับขอบกระดาษในโหมดแสกนเอกสาร (Document Reader)

การตรวจจับขอบกระดาษและวิเคราะห์เอกสารทำงานผ่านกระบวนการหลัก 4 ขั้นตอน ดังนี้:

## 1. การดึงภาพและประมวลผลผ่าน WebAssembly (WASM)
- ระบบจะทำการดึงภาพจากกล้องมาวิเคราะห์อย่างต่อเนื่อง ทุกๆ 0.5 วินาที (500ms) ทำงานในเบื้องหลังผ่าน Hook ที่ชื่อว่า `useDocumentReader`
- ส่งภาพไปให้ไลบรารีชื่อ **`scanic`** ซึ่งเป็นไลบรารี Rust/WASM contour scanner ทำหน้าที่ตรวจหาโครงร่างสี่เหลี่ยมของกระดาษ (Contour Detection) 
- หากพบกระดาษ ระบบจะได้ตำแหน่งมุมทั้ง 4 ของกระดาษกลับมา (Top-Left, Top-Right, Bottom-Right, Bottom-Left)

## 2. การคำนวณความถูกต้องและการจัดมุม (Alignment & Validation)
เมื่อได้พิกัดมุมกระดาษมาแล้ว ระบบจะส่งไปให้ไฟล์ `@/lib/pageEdgeDetection.js` เพื่อคำนวณมิติต่างๆ เพื่อดูว่าพร้อมทำการสแกนหรือยัง:
- **เช็คขอบขาด (Cutoff):** ตรวจสอบว่ามุมของกระดาษชิดขอบจอเกินไปหรือไม่ (เกินขอบเขต Margin 3%) ถ้าชิดเกินไปแปลว่ากระดาษตกขอบ (ขาดซ้าย/ขวา/บน/ล่าง)
- **เช็คระยะห่าง (Coverage):** พื้นที่กระดาษต้องครอบคลุมหน้าจอระหว่าง 15% - 90% ถ้าเล็กไปแปลว่าถือกล้องไกลเกินไป ถ้าใหญ่ไปแปลว่าใกล้เกินไป
- **เช็คจุดกึ่งกลาง (Center Offset):** คำนวณจุดกึ่งกลางกระดาษเปรียบเทียบกับกึ่งกลางหน้าจอ กล้องต้องไม่เบี้ยวไปทางใดทางหนึ่งมากเกินไป (อนุโลมให้คลาดเคลื่อนได้ 12% ของหน้าจอ)
- **เช็คความเอียง (Rotation):** กระดาษต้องไม่เอียงหมุนไปทางซ้ายหรือขวาเกิน 12 องศา
- **เช็คความขนาน (Perspective):** เทียบความยาวของด้านตรงข้ามกัน (ซ้ายเทียบขวา, บนเทียบล่าง) ถ้าระยะห่างต่างกันเกิน 25% แปลว่ากล้องไม่ได้ขนานกับกระดาษ (อาจจะถือกล้องก้มหน้าหรือเงยหน้ามากเกินไป)

## 3. การให้คำแนะนำด้วยเสียง (Audio Guidance)
ระหว่างที่วิเคราะห์ความถูกต้องตามข้อ 2 ระบบจะสร้างคำแนะนำ (Guidance) เป็นภาษาไทยเพื่อบอกผู้ใช้งาน (คนตาบอด) ให้ปรับกล้องแบบเรียลไทม์ เช่น:
- "เอกสารถูกตัดขอบ... ถอยกล้องออก"
- "ขยับเข้าใกล้เอกสารอีกหน่อย"
- "ถอยกล้องออกนิดนึง"
- "เลื่อนกล้องไปทางซ้าย / ขวา / ขึ้นบน / ลงล่าง"
- "เอกสารเอียง หมุนกล้องไปทางซ้าย / ขวา"
- "ก้มกล้อง / เงยกล้องขึ้นให้ขนานกับกระดาษ"

## 4. การถ่ายภาพอัตโนมัติ (Auto-Capture)
- หากระบบพบว่าผู้ใช้ปรับกระดาษเข้ามาอยู่ในตำแหน่งที่ **"ตรงแล้ว" ติดต่อกัน 3 เฟรม** (ประมาณ 1.5 วินาที)
- ระบบจะให้เสียงพูดบอกว่า "ตรงแล้ว กำลังถ่ายเอกสาร" และส่งการสั่นเตือน (Haptic Feedback)
- จากนั้นจะล็อกการทำงานและหยุดสแกนอัตโนมัติ พร้อมทั้งดึงภาพที่ได้ไปส่งให้ AI (Groq Vision) ประมวลผลดึงข้อความ (OCR) สกัดตัวอักษรออกมาอ่านให้ผู้ใช้ฟังต่อไป
